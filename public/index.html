API Basejump: Timestamp microservice

  • Pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
  • If it does contain a date or Unix timestamp, it returns both the Unix timestamp and the natural language form of that date.
  • If it does not contain a date or Unix timestamp, it returns null for those properties.

Example Usage:

http://timestamp-microservice-asell22.c9users.io/February%2026,%202016
http://timestamp-microservice-asell22.c9users.io/1456511945

Example Output:

{ "unix": 1456511945, "natural": "February 26, 2016" }
